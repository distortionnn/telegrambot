Спасибо что приобрели этого бота!
Вот инструкция по установке его на сервер

############################################################################
                          Арендуйте сервера
Если вы хотите установить на сервер не только бота, а еще OutlineVPN, 
то тогда вам понадобится VDS сервер с минимальным объемом HDD 7ГБ.
Если вы планируете что на сервере будет только бот тогда нужно мнинимум 5ГБ HDD.
Процессор и объем оперативной памяти нам не важны выбирайте дешовые варианты.
Операционную систему ставьте Ubuntu 22.

Могу предложить вот таки варианты 
https://ps.profitserver.pro/ - 330р
https://justhost.ru/ - 260р (но объем Hdd слишком маленький встанет или бот или OutlineVPN)
############################################################################
                      Порядок установки на сервер

   /////////////////////////////////////////////////////////////////
                        Подключение к серверу
Используйте любую программу для подключения через ssh, 
рекомендую MobaXterm (https://mobaxterm.mobatek.net/download.html)
После установки:
  1)Нажмите Session
  2)Выберете SSH
  3)В графе Remote host введите ip адрес 
  4)Нажимайте ОК 
  5)В главном меню выбирайте ваш сервер и начнется подключение
  6)Вас попросят вветси logi, вводеите root  
  7)Введите пароль(обычно приходит в письме вместе с ip адресом). 
    Не пугайтесь если не будите ввидеть что вводите, на самом деле все вводится просто скрыто, такая система защиты от любопытных глаз.
   /////////////////////////////////////////////////////////////////
                        Установка бота
1)sudo apt update
2)sudo apt upgrade обновляем пакеты и устанавливаем их
3)sudo apt --reinstall install python3  устанавливаем python3
4)sudo apt --reinstall install python3-pip устанавливаем pip
5)sudo apt install nodejs  устанавливаем Nodejs
6)sudo apt install npm  устанавливаем npm
7)npm install pm2 -g  устанавливаем пакетный менеджер
8)переносим на сервер папку  TgBotVPN из архива
9)cd TgBotVPN переходим в папку
10)pip3 install -r requirenets.txt  устанавливаем все библиотеки бота

11)Откройте файл config.json и заполните все поля
(telegramId можно получить здесь - https://t.me/userinfobot)
Api бота и платежной системы ЮMoney получите с помощью https://t.me/BotFather
API Wallet - https://pay.wallet.tg/
Перейдите в BotFather и выбирете /mybots, вашего бота, Edit Bot здесь вы можете задать картинки и описания для бота
Как настроите перейдите в EditCommands и впишите start - /start после чего у вас в боте появитья кнопка меню с выбором команды start
  
12)pm2 start main.py --interpreter=python3  запускаем бота

После запуска проверьте что все запуситлось в Telegram
Спомощью admin панели добавьте себе любое количество дней
При добавлении нового сервера необязательно указывать IP и Password
подключение происходит по apiUrl и certSha256

pm2 stop main  остановить бота
!!!Внимание если вы хотите изменить что нибудь в файлах text.json или config.json необходимо выключить бота, а после изменений включить!!!
############################################################################
                          Сообщения бота
В файле text.json вы можете написать свои описания для бота.
Что бы перенести текст на новую строку в любом тексте вставьте "\n".
Если вы хотите вставить смайл в ваш текс скопируйте его из телеграма и просто вставьте,
он вставиться в виде символов по типу "\uD83\uD84".
Для того что бы вставить ссылку в текс вставьте <a href="ссылка">текст ссылки</a>
Поля для редактирования:
"hello_message" - Сообщение которое видит пользователь когда первый раз заходит в бота
"trial_message" - Сообщение при начале пробного периода
"about_message" - Сообщение которое появляетя при нажатии на кнопку 'Узнать о VPN'
"how_to_connect_info" - Сообщение при нажатии на кнопку 'Подключить VPN'
"success_pay_message" - Сообщение после успешной оплаты
"ended_sub_message" - Сообщение по истечению срока подписки
"alert_to_renew_sub" - Сообщение о скорои прекращении подписки.
############################################################################
                            Настройки бота
config.json
Поля с конфигурацией 
"admin_tg_id" - напишите сюда ваш id в телеграме
"one_month_cost" - цена одного месяца подписки
"three_month_cost" - цена трех месяцев подписки
"six_month_cost" - цена шести месяцев подписки
"trial_period" - Период пробного времени для новых польлзователей, указывается в секундах 
"UTC_time" - Желаемый часовой пояс
"max_people_server" - Максимальное количество человек на одном серверве
"tg_token" - API токен телеграмм бота 
"tg_shop_token": API токен Юкассы
"tg_wallet_token" API токен TgWallet
############################################################################
              Основные возможности бота для администратора

Вывести пользователей - Показать либо всех пользователей, либо только тех кото купил подписку
Редактировать по id - вводите id пользователя(скопируйте из предыдущего меню) и можете сбросить/добавить вермя подписки
Статичные пользователи - создает/удаляет бессрочный ключ на указанном сервере 
Отправить уведомление - Отправляет введенное сообщение всем пользователям
Сервера - Позволяет добавить/удалить и вывести сервера
      1) Добавить сервер - вводите всю информацию о сервере после чего бот подключается к нему
      2) Удалить сервер - вводите ip сервера, после чего бот удаляет его
      3) Вывести сервера - выводит список всех серверов
